# GW Phoenix 회사 웹사이트

> IT 개발 전문 기업 GW Phoenix의 공식 웹사이트

**라이브**: https://gwPhoenix.github.io/rest02_260608

## 기술 스택

- React 19
- Vite 8
- CSS Variables (커스텀 컬러 시스템)
- GitHub Pages (배포)

## 컬러 시스템

| 역할 | 색상 |
|---|---|
| 메인 배경 (다크블루) | `#0d1b2a` |
| 강조 (로열블루) | `#2a52be` / `#4169e1` |
| 포인트 (다크그린) | `#2d6a42` |
| 포인트 (다크레드) | `#a82424` |

## 개발 실행

```bash
npm install
npm run dev
```

## 배포

```bash
npm run deploy
```
